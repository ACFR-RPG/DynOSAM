Detailed Description

template<typename VALUE>
class dyno::GenericObjectCentricMap< VALUE >

Generic mapping of Object Id -> FrameId -> Value within a nested gtsam::FastMap structure. This is a common datastrcture used for object's to store temporal information about each object.

We call it ObjectCentric map as we order by ObjectId first.
