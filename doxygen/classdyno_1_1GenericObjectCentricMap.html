Detailed Description

template<typename VALUE>
class dyno::GenericObjectCentricMap< VALUE >

Generic mapping of Object Id to FrameId to VALUE within a nested gtsam::FastMap structure. This is a common datastrcture used to store temporal information about each object.

We call it ObjectCentric map as we order by ObjectId first.
