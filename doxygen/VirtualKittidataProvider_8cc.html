 NOTE!!: In vitual kitti, the track id's start at 0 and then the ground truth instances in the semantic instances are indexed from trackId+1 such that a pixel value of zero means not a vehicle. In order to reduce confusion when we work with the semantic images and then associate with the ground truth labels, we will index all tracks as trackID = virtualKittiTrackID + 1 so all tracks start from 1!
