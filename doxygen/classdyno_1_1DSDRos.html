Detailed Description

Shorthand for Dynamic Slam Display ROS.

Facilitates publishing to the following topics:

  • Visual odometry: "odometry"
  • Visual odometry path: "odometry_path"
  • Static point cloud: "static_cloud"
  • Dynamic point cloud: "dynamid_cloud"

Additionally cointains a tf2_ros::TransformBroadcaster to broadcast the visual odometry TF and a DSDTransport to publish the object odometry.

The namespace of the topic is dependant on the effective namespace of the node parsed in via the constructor.
