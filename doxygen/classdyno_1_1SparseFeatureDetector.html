Detects features on the input image using the feature detector.

The features are then spaced out using adaptive non-maxima-supression and refined using subpixel refinement (via cornerSubPix), if enabled.

The number of tracked points indicates how many currently tracked points we currently have; this is used to calculate how many more features we need to reach the minimum number features per frame.
