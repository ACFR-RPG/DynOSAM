Detailed Description

template<typename T = cv::Mat>
class dyno::DataFolder< T >

Defines a set of datafiles within a folder to be laoded by a dataset (partially inspired by pytorch's dataset class). Each folder exists within a parent folder (defined by the dataset) and contains a number of files that need to be loaded by this class. As a number of different input image types (motion/instance segmenation, with and without depth etc.) exist the class is expected to handle the different types itself and return the loaded type.
