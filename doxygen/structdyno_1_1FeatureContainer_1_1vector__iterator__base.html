Detailed Description

template<typename MapIterator, typename MappedType>
struct dyno::FeatureContainer::vector_iterator_base< MapIterator, MappedType >

Internal iterator type allowing iteration over the features directly e.g for(Feature::Ptr : container). This type satisfies constraints for a filter_iterator_base as well as an std::iterator.

Template Parameters
MapIteratorThe internal iterator to use
MappedTypeThe type we are iterating over (either Feature::Ptr or const Feature::Ptr )
