Detailed Description

A spinner that will take a non-blocking function (spinOnce) and puts it in a loop with a given delay. Similar to timer functionality but guarantees that the function is called within a loop. Unlike a timer, the class does not take into account how long the function takes to run as the delay period will be used regardless.

Construct a new Looping Spinner object with a spinOnce style fucntion, name and period (i.e delay). The period is templated on std::chrono::duration<DurationRepT, DurationT> so that chrono literals can be used to specify the delay in timing.

E.g to create a spinner that has a 1 second delay constexpr static std::chrono::second kSecondDelay = 1s; LoopingSpinner spinner(func, "spinner", kSecondDelay);
