Detailed Description

template<typename IMAGETYPE>
struct dyno::ImageWrapper< IMAGETYPE >

ImageWrapper for a ImageType that defines a particular type of input image. The ImageType template is used to create different types (actual c++ types) to differntial cv::Mat's from each other - eg. a type for a RGB image, Optical Flow or Motion Mask.

ImageType must define a static validate(const cv::Mat&) function which throws an expcetion if the cv::Mat does not meed the requirements of the ImageType and a static name() function which returns a string. Must also have a static toRGB(const ImageWrapper<Type>& image) fucntion.
