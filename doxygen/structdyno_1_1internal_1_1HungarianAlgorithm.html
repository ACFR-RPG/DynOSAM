double dyno::internal::HungarianAlgorithm::solve ( const Eigen::MatrixXd &  cost_matrix,
Eigen::VectorXi &  assignment 
) const

Solve the optical assignment problem given a set of workers (row) and jobs (columns). We have to find an assignment of the jobs to the workers, such that each job is assigned to one worker and each worker is assigned one job, such that the total cost of assignment is minimum.

DistMatrix is nonnegative [n * m] matrix (where n <= m), representing n workers and m jobs. The element in the i-th row and j-th column represents the cost of assigning the j-th job to the i-th worker.

The result is a vector of size [n * 1], where each value is the index in the column (j) that i is assigned to such that \hat{j} = assignment[i]
