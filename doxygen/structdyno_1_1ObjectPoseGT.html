Calculates and sets prev_*_current_world_ using ground truth object in the previous frame. Shoudl be used when constructed the gt packets in the various DataProviders.

This object is epxected to be at frame k, and the previous motion should be at frame k-1. The previous_object_gt should have the same object ID and an exception will be thrown if either of these checks fail.

Both ObjectPoseGT are expected to have the L_world_ variable set correctly.
