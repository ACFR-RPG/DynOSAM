dyno::OMDDataLoader::OMDDataLoader ( const fs::path &  dataset_path)

Construct a new Cluster Slam Data Loader object.

Datasset is ordered as per the https://robotic-esp.com/datasets/omd/#tools-id and the dataset path is /path/to/dataset/ /rgbd /raw_depth /vicon.csv .... /optical_flow /instance_masks

The last two folders are not provided in the original dataset and are pre-processed using DynoSAM-Preprocessing
