template<typename ValueTypeT >
ValueTypeT dyno::ParameterDetails::get ( ValueTypeT  default_value) const

Get the c++ value from the node or the default value if param is not set in the node. If the default value is used, the param in the node will NOT be updated.

This default value will only be used if there is NO value in the node, i.e setting a default value from the ParameterConstructor or using paramter overwrites using NodeOptions or the launchfile will set value in the node and the default_value argument will not be used.
