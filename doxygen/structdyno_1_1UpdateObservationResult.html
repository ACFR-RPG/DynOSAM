Detailed Description

Result of a static/dynamic point update by calling Formulation<MAP>::updateStaticObservations or Formulation<MAP>::updateDynamicObservations.

The objects affected per frame should be specified and any additional paramters for the optimisation (i.e gtsam::ISAM2UpdateParams) can be set, which will then be parsed to the optimiser by the BackendModule.
