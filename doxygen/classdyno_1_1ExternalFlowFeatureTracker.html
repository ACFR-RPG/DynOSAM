Base function to perform feature tracking on static points.

Is not intended to be state-ful (ie. contains a reference to the last frame) and instead we pass in the previous frame everytime.

If the previous frame is null this should indicate that this is the first frame to be tracked!

Parameters
previous_frameFrame::Ptr. Previous frame (k-1) with filled-out features to be tracked.
image_containerconst ImageContainer&. Contains current (k) images which will be tracked from the previous frame.
tracker_infoFeatureTrackerInfo&. Tracking metadata to be filled out.
detection_maskconst cv::Mat& A detection mask in the opencv feature tracking form: CV_8UC1 where white pixels (255) are valid and black pixels (0) should not be detected on
Returns
FeatureContainer Contains all successsfully tracked features.
