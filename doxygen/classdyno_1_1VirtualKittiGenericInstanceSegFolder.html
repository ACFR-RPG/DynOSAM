inlineprotected

Loads instance semantic mask from the file path.

This loads the image using libpng++ since we need to load the image as an 8bit indexed PNG file (which opencv does not have functionality for). Keeping consistent with the ground truth indexing we use for VirtualKitti we do not need to modify the trackID's within the image since we want the trackID's to be indexed from 1
