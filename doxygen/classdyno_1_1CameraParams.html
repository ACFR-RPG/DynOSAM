Constructs the parameters for a specific camera.

Distortion model expects a string as either "none", "plumb_bob", "radial-tangential", "radtan", "equidistant" or "kanna_brandt". The corresponding DistortionModel (enum) will then be assigned.

Parameters
intrinsicsconst IntrinsicsCoeffs& Coefficients for the intrinsics matrix. Should be in the form [fu fv cu cv]
distortionconst DistortionCoeffs& Coefficients for the camera distortion.
image_sizeconst cv::Size& Image width and height
distortion_model_const std::string& Expected distortion model to be used for this camera.
T_robot_cameraconst gtsam::Pose3& The camera extrinsics describing the transformation of the camera in the robot frame
