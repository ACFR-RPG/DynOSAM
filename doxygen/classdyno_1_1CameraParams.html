Constructs the parameters for a specific camera.

Distortion model expects a string as either "none", "plumb_bob", "radial-tangential", "radtan", "equidistant" or "kanna_brandt". The corresponding DistortionModel (enum) will then be assigned.

Converts the DistortionModel enum to a distortion model string and associated camera model.

The returned distortion model (e.g. "plumb_bob", "radtan") will match distortion model's used in the ROS ecosystem which are defined in sensor_msgs/distortion_models.hpp.

Only pinhole camera models are currently supported.

Out-args are only set when the function returns true.
