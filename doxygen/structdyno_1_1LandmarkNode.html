Detailed Description

template<typename MEASUREMENT>
class dyno::LandmarkNode< MEASUREMENT >

Landmark node representing all the measurements of a particular landmark (i) over all time (k to K) in the map-graph.

The node can represent either a static or dynamic landmark and has multiple measurements associated with it.
